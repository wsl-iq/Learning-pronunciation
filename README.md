### **About the Project**

English Learning Pronunciation is an educational application designed to help learners improve their English skills, especially in pronunciation, vocabulary, and phonetics.

The program focuses on making learning clear, practical, and interactive, suitable for students and self-learners.

---

### **Purpose**

The main goal of this application is to:
- Improve English pronunciation
- Help users understand phonetic symbols (IPA)
- Build a strong vocabulary base
- Provide a simple training and testing environment

---

### **Who Is This For?**

This program is useful for:
- Students (especially English learners)
- Teachers (as a supporting tool)
- Self-learners
- Anyone who wants to improve pronunciation and speaking skills

---

### **Key Features**

### **1. Trained Automatic Pronunciation**
- The program includes an automatic pronunciation system
- Words are pronounced clearly using trained voice logic
- Helps users learn the correct accent and sound

### **2. Phonetic Transcription (IPA)**
- Each word includes its phonetic transcription
- Users can understand how words are pronounced scientifically

- **Example**

| Word   | Phonetic Transcription (IPA) |
|--------|-----------------------------|
| Hello  | /həˈloʊ/                    |
| World  | /wɜːrld/                    |
| Apple  | /ˈæpəl/                     |
| Book   | /bʊk/                       |
| Cat    | /kæt/                       |
| Dog    | /dɒɡ/                       |
| Fish   | /fɪʃ/                       |

### 3. Vocabulary Learning
- Organized list of English words
- Easy to browse and study
- Focus on real usage

### **4. Training Mode**
- Learn words step-by-step
- Listen, read, and understand pronunciation

### **5. Testing Mode**
- Test your knowledge
- Improve memory and recognition
- Evaluate your progress

### **6. Phonetic Dictionary**
- Special section for learning sounds (IPA)
- Helps understand vowels, consonants, and sound differences

### **7. Pronunciation Support**
- Focus on sound clarity
- Letter differences (like `f` / `v`)
- Real speaking improvement

### **8. Simple and Clean Interface**
- Easy to use
- No complexity
- Suitable for beginners

---

### **How to Use**

1. Open the program
2. Choose a section: Training, Test, or Phonetic Dictionary
3. Select a word or lesson
4. Listen to pronunciation
5. Read the phonetic transcription
6. Repeat and practice
7. Test yourself

### **Coding Languages and Tools Used**

- **Python** : Only to run the program
- **HTML, CSS, JavaScript** : For the web-based interface
- **JSON** : For local data storage

`JSON` is used to store vocabulary data, user progress, and settings locally on the user's machine. This allows the program to save and retrieve information without needing an internet connection, ensuring smooth and efficient operation.

```json
{
  "word": "example",
  "ipa": "/ɪɡˈzæmpəl/"
}
```
